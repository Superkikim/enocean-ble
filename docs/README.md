# Documentation

This folder contains the project documentation for the Home Assistant custom
integration `enocean_ble`.

## Contents

- `PROJECT.md`: scope, supported hardware, capabilities, limits.
- `ARCHITECTURE.md`: runtime architecture and main code paths.
- `CONFIG_FLOW.md`: commissioning flow behavior and state machine.
- `OPERATIONS.md`: local runbook (logs, deploy, validation).
- `TROUBLESHOOTING.md`: common issues and checks.
- `ROADMAP.md`: high-level delivery status and next steps.
